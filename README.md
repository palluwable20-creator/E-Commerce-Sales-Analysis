# E-Commerce-Sales-Analysis
Python and Power BI project for analyzing e-commerce sales and website performance.

## Project Overview
This project analyzes e-commerce sales and website performance using Python and Power BI.

## Tools Used
- Python
-Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI
- GitHub

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Interactive Power BI Dashboard
- Business Insights

## 🎯 Objectives

- Analyze sales performance
- Identify top-performing regions
- Compare product category sales
- Evaluate customer segments
- Analyze payment methods
- Study repeat customer behavior
- Monitor website visits and revenue

## Key Insights
- Revenue by Region
- Revenue by Product Category
- Website Visits Analysis
- Customer Segment Analysis
- Payment Mode Analysis
- Repeat Customer Analysis
